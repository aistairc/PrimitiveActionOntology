# PrimitiveActionOntology
A primitive action satisfies the following criteria

1. An action changes or maintains the state of the object of the action
2. An action is conscious.
3. An action is not classified according to the type of the object of the action
4. An action does not include the interpretation of the observer.
5. An action is not composed of several primitive actions.

The first point means that it does not include how the action is realised. For example, when we think of the action "cut", we cannot say "cut by scissors" or "cut by knife". For example, when we think of the act of "cut", if we include what is used to cut the object, such as "cut by scissors" or "cut by knife", our vocabulary will be unlimited, depending on the combination of the state change of the object and the way to realise it. In this example, we extract only the part that expresses the state change, without regard to the method, and use only "cut" as a primitive action.
The second point is that only conscious actions are considered, and unconscious actions such as "sleep" and "wake" are considered as "lay down" and "sit", respectively. Unconscious actions such as "sleep" and "wake" are represented by "lay down" and "sit", respectively.
The third criterion is not to distinguish between different actions depending on what the object is. For example, if the object of the act of "grasp" is a "cup" or an "upper limb", then For example, when the object of the "grasp" action is a "cup" or an "upper limb", there is no primitive action called "grasp cup" or "grasp upper limb", respectively.
The fourth criterion is to exclude actions that require interpretation by the observer. For example, only the executor of the action "think" knows whether it is being performed or not, and the observer cannot directly recognise it. We exclude such actions from the list of primitive actions.
The fifth criterion is to exclude actions that multiple primitive actions can express. For example, the action "buy" can be represented by "pick up money," "carry money," and "carry item. For example, the action "buy" can be expressed as a series of primitive actions such as "pick up money," "carry money," and "carry item. Such an action is not considered a primitive action.
